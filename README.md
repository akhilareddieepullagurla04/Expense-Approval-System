# Expense Approval System

An Expense Approval System built to manage employee expense submissions, approvals, and tracking in a simple and organized way.

## Overview

This project helps employees submit expense requests such as travel, food, accommodation, office supplies, and other business-related expenses. Managers or admins can review the submitted requests and approve or reject them.

The system reduces manual paperwork, improves tracking, and makes the approval process faster and more transparent.

## Features

- Employee expense submission
- Expense category selection
- Amount entry and purpose/description
- View submitted expense requests
- Approve or reject requests
- Status tracking
- Centralized expense record management

## Modules

### Employee Module
- Submit new expense request
- Enter expense details
- Select expense category
- View own submitted expenses
- Track approval status

### Admin / Manager Module
- View all submitted expenses
- Review expense details
- Approve or reject requests
- Monitor overall expense records

## Technologies Used

## Project Type

This is a web-based mini project for expense management and approval.

If AI or NLP is included for classifying expense descriptions, it can be described as:
- AI-enabled Expense Approval System
- ML-based Expense Approval System

## How It Works

1. The employee enters expense details.
2. The request is stored in the system.
3. The admin or manager views all submitted requests.
4. The admin approves or rejects the expense.
5. The employee can later check the request status.
